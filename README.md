# Yatube
Социальная сеть для блогеров на Django

# Описание
Социальная сеть для публикации личных записей. Это сайт, на котором можно создать свою страницу. Если на нее зайти, то можно посмотреть все записи автора. Пользователи могут заходить на чужие страницы, комментировать их записи. Автор может выбрать имя и уникальный адрес для своей страницы. Администратор имеет возможность модерировать записи и блокировать пользователей. Записи можно отправить в группу и посмотреть в ней записи разных авторов.
